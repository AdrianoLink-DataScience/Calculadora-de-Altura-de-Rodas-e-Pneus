# 🚗 Simulador de Fitment e Geometria de Rodas

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_svg.svg)](https://calculadora-de-altura-de-rodas-e-pneus.streamlit.app/)

Este projeto é uma ferramenta interativa desenvolvida para **feiras e eventos automotivos**. O objetivo é permitir que entusiastas e profissionais simulem, em tempo real, como a troca de rodas e pneus altera a geometria, a estética (*stance*) e a altura do chassi em relação ao solo.

## 🌟 Funcionalidades

- **Cálculo de Diâmetro Real:** Converte as medidas nominais do pneu (ex: 225/45 R18) para dimensões físicas exatas em milímetros e centímetros.
- **Comparação Visual:** Gráficos gerados via Matplotlib que comparam o setup atual com o novo projeto em escala real.
- **Análise de Altura:** Mostra exatamente quantos centímetros o carro ficará mais alto ou mais baixo do solo.
- **Interface Showroom:** Layout otimizado para exibição em notebooks e telas grandes, focando em métricas visuais claras.

## 🛠️ Tecnologias Utilizadas

- **Python 3.10+**
- **Streamlit:** Interface web interativa.
- **Matplotlib:** Renderização dos gráficos geométricos.
- **NumPy:** Cálculos matemáticos de precisão.
